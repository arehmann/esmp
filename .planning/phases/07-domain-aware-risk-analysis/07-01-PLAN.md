3. **computeDomainCriticality()** (DRISK-01) — Uses pattern comprehension to find USES_TERM edges to BusinessTerm nodes with High criticality. If any linked term has criticality="High", domainCriticality=1.0. Map "Medium" to 0.5 for future-proofing. Classes with no USES_TERM edges get 0.0.
   ```java
   private void computeDomainCriticality() {
       String cypher = """
           MATCH (c:JavaClass)
           WITH c,
                size([(c)-[:USES_TERM]->(t:BusinessTerm) WHERE t.criticality = 'High' | t]) AS highCount,
                size([(c)-[:USES_TERM]->(t:BusinessTerm) WHERE t.criticality = 'Medium' | t]) AS medCount
           SET c.domainCriticality = CASE
               WHEN highCount > 0 THEN 1.0
               WHEN medCount > 0 THEN 0.5
               ELSE 0.0 END
           """;
       neo4jClient.query(cypher).run();
   }
   ```

4. **computeSecuritySensitivity()** (DRISK-02) — Graduated heuristic: name keyword match=0.3, annotation match=0.5, both=+0.2 bonus, package match=+0.2 boost. Clamped to [0.0, 1.0] via min(1.0, ...).
   Use `toLower(coalesce(c.simpleName, '')) =~ $namePattern` for name matching.
   Use `ANY(a IN coalesce(c.annotations, []) WHERE toLower(a) =~ $annotPattern)` for annotation matching.
   Use `toLower(coalesce(c.packageName, '')) =~ $pkgPattern` for package matching.
   Pass all patterns and weights as Cypher parameters (no string interpolation in Cypher).

5. **computeFinancialInvolvement()** (DRISK-03) — Same graduated pattern as security, plus USES_TERM boost. Check if class has USES_TERM edge to a BusinessTerm whose termId or displayName matches financial keywords. Use `EXISTS { MATCH (c)-[:USES_TERM]->(t:BusinessTerm) WHERE toLower(coalesce(t.termId, '')) =~ $termPattern OR toLower(coalesce(t.displayName, '')) =~ $termPattern }` for the term hit. Financial annotations are unlikely in Java, so the annotPattern can reuse an empty-match pattern or financial keywords can omit annotation matching entirely. Use `'(?!x)x'` (never-match pattern) for annotPattern if no financial-specific annotations exist.

6. **computeBusinessRuleDensity()** (DRISK-04) — Log-normalized count of DEFINES_RULE edges using pattern comprehension:

   private void computeBusinessRuleDensity() {
       String cypher = """
           MATCH (c:JavaClass)
           WITH c,
                size([(c)-[:DEFINES_RULE]->(t) | t]) AS ruleCount
           SET c.businessRuleDensity = log(1.0 + ruleCount)
           """;

   }

7. **computeEnhancedRiskScore()** (DRISK-05) — Single Cypher query combining all 8 dimensions from raw properties. Does NOT use structuralRiskScore as input (avoids double-weighting). Uses the 8 enhanced weights from RiskWeightConfig.
   ```java
   private void computeEnhancedRiskScore() {
       String cypher = """
           MATCH (c:JavaClass)
           SET c.enhancedRiskScore = (
               $wComplexity  * log(1.0 + coalesce(c.complexitySum, 0)) +
               $wFanIn       * log(1.0 + coalesce(c.fanIn, 0)) +
               $wFanOut      * log(1.0 + coalesce(c.fanOut, 0)) +
               $wDbWrites    * CASE WHEN coalesce(c.hasDbWrites, false) THEN 1.0 ELSE 0.0 END +
               $wDomainCrit  * coalesce(c.domainCriticality, 0.0) +
               $wSecurity    * coalesce(c.securitySensitivity, 0.0) +
               $wFinancial   * coalesce(c.financialInvolvement, 0.0) +
               $wRuleDensity * coalesce(c.businessRuleDensity, 0.0)
           )
           """;
       neo4jClient.query(cypher)
           .bindAll(Map.of(
               "wComplexity",  riskWeightConfig.getDomainComplexity(),
               "wFanIn",       riskWeightConfig.getDomainFanIn(),
               "wFanOut",      riskWeightConfig.getDomainFanOut(),
               "wDbWrites",    riskWeightConfig.getDomainDbWrites(),
               "wDomainCrit",  riskWeightConfig.getDomainCriticality(),
               "wSecurity",    riskWeightConfig.getSecuritySensitivity(),
               "wFinancial",   riskWeightConfig.getFinancialInvolvement(),
               "wRuleDensity", riskWeightConfig.getBusinessRuleDensity()))
           .run();
   }
   ```
