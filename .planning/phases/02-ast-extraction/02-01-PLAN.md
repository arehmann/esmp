       - Add libraries: `openrewrite-java` (org.openrewrite:rewrite-java), `openrewrite-java-21` (org.openrewrite:rewrite-java-21), `vaadin-server` (com.vaadin:vaadin-server) — all using version.ref
       - Add vaadin-server as testImplementation only (provides Vaadin 7 type symbols for classpath resolution in test fixtures)

    2. **Add to build.gradle.kts:**
       - `implementation(libs.openrewrite.java)` and `implementation(libs.openrewrite.java21)`
       - `testImplementation(libs.vaadin.server)`
       - Run `./gradlew dependencies` to verify no classpath conflicts with Spring Boot BOM. If conflicts found, use `exclude` groups or `resolutionStrategy.force` per RESEARCH.md Pitfall 3.

    3. **Create Neo4j @Node entities in com.esmp.extraction.model:**

       **ClassNode.java** — `@Node("JavaClass")`:
       - `@Id String fullyQualifiedName` (business key, never changes)
       - `@Version Long version` (enables SDN MERGE detection for idempotent save)
       - `@DynamicLabels Set<String> extraLabels` (for VaadinView, VaadinComponent, VaadinDataBinding secondary labels per user decision)
       - Properties: `simpleName`, `packageName`, `List<String> annotations`, `List<String> modifiers`, `List<String> imports`, `boolean isInterface`, `boolean isAbstract`, `boolean isEnum`, `String superClass`, `List<String> implementedInterfaces`, `String sourceFilePath`, `String contentHash` (SHA-256 of source for idempotency/change detection)
       - `@Relationship(type = "DECLARES_METHOD", direction = OUTGOING) List<MethodNode> methods`
       - `@Relationship(type = "DECLARES_FIELD", direction = OUTGOING) List<FieldNode> fields`
       - `@Relationship(type = "CONTAINS_COMPONENT", direction = OUTGOING) List<ContainsComponentRelationship> componentChildren` (Vaadin layout hierarchy)

       **MethodNode.java** — `@Node("JavaMethod")`:
       - `@Id String methodId` (format: `com.example.MyClass#myMethod(String,int)` — unique per overload)
       - `@Version Long version`
       - Properties: `simpleName`, `String returnType`, `List<String> parameterTypes`, `List<String> annotations`, `List<String> modifiers`, `boolean isConstructor`, `String declaringClass` (FQN for reverse lookup)
       - `@Relationship(type = "CALLS", direction = OUTGOING) List<CallsRelationship> callsOut` (call graph edges)

       **FieldNode.java** — `@Node("JavaField")`:
       - `@Id String fieldId` (format: `com.example.MyClass#fieldName`)
       - `@Version Long version`
       - Properties: `simpleName`, `String fieldType`, `String declaringClass`, `List<String> annotations`, `List<String> modifiers`

       **CallsRelationship.java** — `@RelationshipProperties` for CALLS edge:
       - `@Id @GeneratedValue Long id` (relationship IDs are internal, not business keys)
       - `@TargetNode MethodNode target`
       - Properties: `String callerMethodId`, `String callSiteFile`, `int callSiteLine`

       **ContainsComponentRelationship.java** — `@RelationshipProperties` for CONTAINS_COMPONENT edge:
       - `@Id @GeneratedValue Long id`
       - `@TargetNode ClassNode child` (the child component class)
       - Properties: `String parentComponentType`, `String childComponentType`

    4. **Create ExtractionConfig.java** — `@Configuration @ConfigurationProperties("esmp.extraction")`:
       - `String sourceRoot` (default source path per user decision: `esmp.target.source-root` config property — note: use `esmp.extraction.source-root` to stay within extraction namespace)
       - `String classpathFile` (path to pre-exported classpath file from target project)

    5. **Create Neo4jSchemaInitializer.java** — `@Component` implementing `ApplicationRunner`:
       - On startup, execute Cypher statements via `Neo4jClient` (not repository) to create uniqueness constraints:
         ```
         CREATE CONSTRAINT java_class_fqn_unique IF NOT EXISTS FOR (n:JavaClass) REQUIRE n.fullyQualifiedName IS UNIQUE
         CREATE CONSTRAINT java_method_id_unique IF NOT EXISTS FOR (n:JavaMethod) REQUIRE n.methodId IS UNIQUE
         CREATE CONSTRAINT java_field_id_unique IF NOT EXISTS FOR (n:JavaField) REQUIRE n.fieldId IS UNIQUE
         ```
       - Log constraint creation at INFO level
       - Use `IF NOT EXISTS` so it's safe to run repeatedly

    6. **Update application.yml** — add extraction config defaults:
       ```yaml
       esmp:
         extraction:
           source-root: ""
           classpath-file: ""
       ```

    7. **Verify compilation:** Run `./gradlew compileJava` — must succeed with no dependency conflicts. If OpenRewrite pulls conflicting Jackson/ASM versions, resolve per RESEARCH.md guidance (exclude or force Spring Boot managed versions).
