1. **MigrationToolService.java** — Create `src/main/java/com/esmp/mcp/tool/MigrationToolService.java`:
   - `@Component`
   - Constructor-inject: `MigrationContextAssembler`, `GraphQueryService`, `RiskService`, `LexiconService`, `VectorSearchService`, `ValidationService`, `MeterRegistry`
   - **6 methods, each annotated with `@Tool(description="...")`:**

   a. `getMigrationContext(String classFqn)`:
      - `@Tool(description = "Retrieves comprehensive migration context for a Java class. Input: fully-qualified class name (e.g. com.example.PaymentService). Returns: dependency cone, code chunks, business terms, risk analysis, business rules. Use as primary context tool before migrating any class.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "getMigrationContext"})`
      - Delegates to `assembler.assemble(classFqn)`
      - Increments `Counter.builder("esmp.mcp.tool.invocations").tag("tool", "getMigrationContext").register(meterRegistry).increment()`
      - Returns `MigrationContext`
      - Logs structured request: `log.info("MCP_REQUEST tool=getMigrationContext classFqn={} latencyMs={} completeness={} truncated={} warnings={}", ...)`

   b. `searchKnowledge(String query, String module, String stereotype, int topK)`:
      - `@Tool(description = "Semantic vector search across indexed code chunks. Input: query string + optional filters (module, stereotype, topK). Returns: ranked code chunks with similarity scores. Use for finding relevant code when you don't have a specific class FQN.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "searchKnowledge"})`
      - Creates `SearchRequest(query, module, stereotype, null, topK > 0 ? topK : 10)`
      - Delegates to `vectorSearchService.search(request)`
      - Returns `SearchResponse` (from vector API)

   c. `getDependencyCone(String classFqn, int maxDepth)`:
      - `@Tool(description = "Explores the dependency graph for a class. Input: class FQN + optional depth (default 10). Returns: all transitively reachable nodes via DEPENDS_ON, EXTENDS, IMPLEMENTS, CALLS, BINDS_TO, QUERIES, MAPS_TO_TABLE edges. Use to understand what a class depends on.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "getDependencyCone"})`
      - `@Cacheable(value = "dependencyCones", key = "#classFqn")`
      - Delegates to `graphQueryService.findDependencyCone(classFqn)`
      - Returns `Optional<DependencyConeResponse>` (null if class not found)

   d. `getRiskAnalysis(String classFqn, String module, String sortBy, int limit)`:
      - `@Tool(description = "Risk analysis: if classFqn provided, returns detailed risk for that class (complexity, fan-in/out, domain criticality, security, financial). If classFqn is empty/null, returns risk heatmap filterable by module and sortable by 'structural' or 'enhanced'. Use when you need risk-specific info.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "getRiskAnalysis"})`
      - If `classFqn` is not blank: delegate to `riskService.getClassDetail(classFqn)`, return the `Optional<RiskDetailResponse>`
      - If `classFqn` is blank: delegate to `riskService.getHeatmap(module, null, null, limit > 0 ? limit : 20, sortBy != null ? sortBy : "enhanced")`, return the `List<RiskHeatmapEntry>`
      - Return type is `Object` (either RiskDetailResponse or List<RiskHeatmapEntry>)

   e. `browseDomainTerms(String search, String criticality)`:
      - `@Tool(description = "Browse or search the domain lexicon. Input: optional search text and/or criticality filter (High, Medium, Low). Returns: business terms with definitions, related classes, usage counts. Use to understand business terminology in the codebase.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "browseDomainTerms"})`
      - `@Cacheable(value = "domainTermsByClass", key = "#search + '_' + #criticality")`
      - Delegates to `lexiconService.findByFilters(search, criticality, null)`
      - Returns `List<BusinessTermResponse>`

   f. `validateSystemHealth()`:
      - `@Tool(description = "Runs all validation queries against the knowledge graph and vector store. Returns: pass/fail/warn counts and detailed results for each query. Call before starting migration to confirm data integrity.")`
      - `@Timed(value = "esmp.mcp.request.duration", extraTags = {"tool", "validateSystemHealth"})`
      - Delegates to `validationService.runAll()`
      - Returns `ValidationReport`

   - All methods include structured logging: `log.info("MCP_REQUEST tool={} params={} latencyMs={}", ...)`

2. **McpToolRegistration.java** — Create `src/main/java/com/esmp/mcp/config/McpToolRegistration.java`:
   ```java
   package com.esmp.mcp.config;

   import com.esmp.mcp.tool.MigrationToolService;
   import org.springframework.ai.tool.ToolCallbackProvider;
   import org.springframework.ai.tool.method.MethodToolCallbackProvider;
   import org.springframework.context.annotation.Bean;
   import org.springframework.context.annotation.Configuration;

   @Configuration
   public class McpToolRegistration {

       @Bean
       public ToolCallbackProvider migrationTools(MigrationToolService toolService) {
           return MethodToolCallbackProvider.builder()
               .toolObjects(toolService)
               .build();
       }
   }
   ```
