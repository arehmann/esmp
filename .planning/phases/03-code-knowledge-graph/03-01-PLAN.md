  <action>
    Create four new @Node entity classes following the exact ClassNode pattern:

    **AnnotationNode** (@Node("JavaAnnotation")):
    - @Id fullyQualifiedName (String, e.g., "org.springframework.stereotype.Service")
    - @Version Long version
    - simpleName (String)
    - packageName (String)
    - retention (String, nullable — "RUNTIME", "CLASS", "SOURCE" if parsed)
    - No-arg + FQN constructor, getters/setters

    **PackageNode** (@Node("JavaPackage")):
    - @Id packageName (String, e.g., "com.example.sample")
    - @Version Long version
    - simpleName (String — last segment e.g., "sample")
    - moduleName (String — owning module name)
    - @Relationship(type = "CONTAINS_CLASS", direction = OUTGOING) List<ClassNode> classes
    - No-arg + packageName constructor, getters/setters

    **ModuleNode** (@Node("JavaModule")):
    - @Id moduleName (String — Gradle subproject name or source root dir name)
    - @Version Long version
    - sourceRoot (String — absolute path)
    - isMultiModuleSubproject (boolean)
    - @Relationship(type = "CONTAINS_PACKAGE", direction = OUTGOING) List<PackageNode> packages
    - No-arg + moduleName constructor, getters/setters

    **DBTableNode** (@Node("DBTable")):
    - @Id tableName (String, lowercased)
    - @Version Long version
    - schemaName (String, nullable)
    - No-arg + tableName constructor, getters/setters

    Create six @RelationshipProperties classes following the CallsRelationship pattern:

    **DependsOnRelationship**: @Id @GeneratedValue Long id, @TargetNode ClassNode target, String injectionType ("field", "constructor", "setter"), String fieldName
    **ExtendsRelationship**: @Id @GeneratedValue Long id, @TargetNode ClassNode target, String resolutionConfidence ("RESOLVED" or "UNRESOLVED")
    **ImplementsRelationship**: @Id @GeneratedValue Long id, @TargetNode ClassNode target, String resolutionConfidence
    **BindsToRelationship**: @Id @GeneratedValue Long id, @TargetNode ClassNode target, String bindingMechanism (e.g., "BeanFieldGroup", "FieldGroup")
    **QueriesRelationship**: @Id @GeneratedValue Long id, @TargetNode DBTableNode target, String queryType ("derived" or "explicit")
    **MapsToTableRelationship**: @Id @GeneratedValue Long id, @TargetNode DBTableNode target

    All relationship classes: no-arg constructor + parameterized constructor, getters/setters.

    Create four new repository interfaces following the ClassNodeRepository pattern:
    - AnnotationNodeRepository extends Neo4jRepository<AnnotationNode, String>
    - PackageNodeRepository extends Neo4jRepository<PackageNode, String>
    - ModuleNodeRepository extends Neo4jRepository<ModuleNode, String>
    - DBTableNodeRepository extends Neo4jRepository<DBTableNode, String>

    Extend Neo4jSchemaInitializer.run() to add four new constraints:
    - java_annotation_fqn_unique for JavaAnnotation.fullyQualifiedName
    - java_package_name_unique for JavaPackage.packageName
    - java_module_name_unique for JavaModule.moduleName
    - db_table_name_unique for DBTable.tableName

    All classes in package com.esmp.extraction.model (entities/relationships) and com.esmp.extraction.persistence (repositories).
