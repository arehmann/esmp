Create ChunkingService as a `@Service` in `com.esmp.vector.application`. Inject `Neo4jClient`.

**chunkClasses(String sourceRoot) method:**
1. Query Neo4j for all JavaClass nodes with non-null sourceFilePath:
   ```cypher
   MATCH (c:JavaClass) WHERE c.sourceFilePath IS NOT NULL
   RETURN c.fullyQualifiedName AS fqn, c.simpleName AS simpleName, c.packageName AS pkg,
          c.sourceFilePath AS sourcePath, c.contentHash AS hash,
          c.structuralRiskScore AS srs, c.enhancedRiskScore AS ers,
          c.domainCriticality AS dc, c.securitySensitivity AS ss,
          c.financialInvolvement AS fi, c.businessRuleDensity AS brd,
          labels(c) AS labels
   ```
2. For each class, read source file from disk (guard: `Files.exists(Path.of(sourcePath))`; skip if missing).
3. Query 1-hop neighbors via Neo4jClient:
   ```cypher
   MATCH (c:JavaClass {fullyQualifiedName: $fqn})
   OPTIONAL MATCH (caller:JavaClass)-[:DEPENDS_ON]->(c)
   OPTIONAL MATCH (c)-[:DEPENDS_ON]->(dep:JavaClass)
   OPTIONAL MATCH (c)-[:IMPLEMENTS]->(iface:JavaClass)
   OPTIONAL MATCH (c)-[:USES_TERM]->(t:BusinessTerm)
   OPTIONAL MATCH (c)-[:DECLARES_METHOD]->(m:JavaMethod)
   RETURN collect(DISTINCT caller.fullyQualifiedName) AS callers,
          collect(DISTINCT dep.fullyQualifiedName) AS dependencies,
          collect(DISTINCT iface.fullyQualifiedName) AS implementors,
          collect(DISTINCT {termId: t.termId, displayName: t.displayName}) AS terms,
          collect(DISTINCT m.methodId) AS methodIds
   ```
4. Query callees (classes this class's methods call):
   ```cypher
   MATCH (c:JavaClass {fullyQualifiedName: $fqn})-[:DECLARES_METHOD]->(m:JavaMethod)-[:CALLS]->(target:JavaMethod)
   RETURN collect(DISTINCT target.declaringClass) AS callees
   ```
5. Detect Vaadin 7: check if labels list contains "VaadinView", "VaadinComponent", or "VaadinDataBinding". Set `vaadin7Detected = true` and collect matching labels as `vaadinPatterns`.
6. Determine primary stereotype from labels (first match of: Service, Repository, VaadinView, Controller, or empty string).
7. Determine module from packageName (second segment after "com.esmp." or the full packageName).

**Build CLASS_HEADER chunk:**
- pointId = `ChunkIdGenerator.chunkId(fqn, "__HEADER__")`
- text = `"[CLASS: {simpleName}] [PACKAGE: {packageName}]\n{classJavadoc from source}\n{field declarations from source}"`
  - Extract class-level javadoc and field declarations by parsing source text between class opening brace and first method. Use simple string extraction (find `class ... {` then collect lines until first method signature pattern).
- All enrichment fields populated from the class-level query results.

**Build METHOD chunks (one per methodId):**
- For each methodId, extract method source from the file using simple regex/string matching (find method signature in source text, extract body including braces).
- pointId = `ChunkIdGenerator.chunkId(fqn, methodSignature)` where methodSignature is the part after `#` in methodId.
- classHeaderId = header chunk's pointId.toString()
- text = `"[CLASS: {simpleName}] [METHOD: {methodSignature}]\n{method body source}"`
- Enrichment same as class (risk scores, neighbors, terms) since enrichment is class-level.

Return `List<CodeChunk>` (all header + method chunks for all classes).
